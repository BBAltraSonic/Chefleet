# Plus Jakarta Sans Font Setup

## Download Instructions

Download Plus Jakarta Sans font from Google Fonts:
https://fonts.google.com/specimen/Plus+Jakarta+Sans

## Required Font Files

Place the following font files in this directory:
- `PlusJakartaSans-Regular.ttf` (weight: 400)
- `PlusJakartaSans-Medium.ttf` (weight: 500)
- `PlusJakartaSans-Bold.ttf` (weight: 700)
- `PlusJakartaSans-ExtraBold.ttf` (weight: 800)

## Quick Download

1. Visit https://fonts.google.com/specimen/Plus+Jakarta+Sans
2. Click "Download family"
3. Extract the ZIP file
4. Copy the required .ttf files from the `static/` folder to this directory
5. Run `flutter pub get` to update dependencies
